# Finance Tracker Frontend

React frontend for the Personal Finance Tracker application.

## Quick Start

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env` file:
```env
VITE_API_URL=http://localhost:8000/api
```

## Build for Production

```bash
npm run build
```

## Features

- User Authentication (Login/Register)
- Google OAuth Integration
- Transaction Management
- Budget Tracking
- AI Financial Advisor
- Bank Statement Upload
- Receipt Management
- Financial Dashboard

## Tech Stack

- React 18
- Vite
- Axios
- Modern CSS
