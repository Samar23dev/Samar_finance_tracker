## Features

- User Authentication & Google OAuth
- Transaction Management (Income, Expenses, Transfers)
- Budget Goals & Tracking
- AI Financial Advisor (Google Gemini)
- Bank Statement Upload (PDF/CSV) with Auto-Transaction Creation
- Financial Dashboard with Charts
- Email Notifications

## Tech Stack

**Backend**: Django, PostgreSQL, Google Gemini AI  
**Frontend**: React, Vite  
**Deployment**: Render

## Quick Start

### Backend
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend
```bash
cd finance-tracker-frontend
npm install
npm run dev
```
